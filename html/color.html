Material theme in Android offers a way to customize a look of application by defining a color palette that represents the style of application. By changing only color palette attributes, the color properties of multiple widgets will be affected.  For example, if application specifies the value for "ColorPrimary" attribute, all widget resources that refer to "ColorPrimary" will be changed to the given value.
